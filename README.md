## Ogólne
This project is simple Lorem ipsum dolor generator.
	
## Technologie
Projekt działa na : 
* Python : 3.8

Wymagane biblioteki : 
* PyQt5
* pymongo
* numpy
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../lorem
$ npm install
$ npm start
```
